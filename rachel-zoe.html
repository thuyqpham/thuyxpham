A box subscription e-commerce

UX & UI design
design exploration

Client is a celebrity that sells subscriptions of gift boxes.  Current site caused major confusion for users and also required branding face lift.

redesign

Current site did not convey key ideas which created confusion for users. Main confusion was that some users thought they were buying a box and not a recurring subscription. Some items required user to select from choices while other items were pre-selected.  User flow to arrive at checkout required too many steps that caused abandonment.

Client did not think current design fit the brand image of luxury and wanted a face lift of visual design.

  • Clarify product as subscription
  • Improve order flow
  • Visual identity re-design
tools used

Sketch used for design & wireframes. Invision used for prototyping.
