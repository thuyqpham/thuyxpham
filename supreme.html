A web app for fashion design team

UX & UI design

Client is a big global fashion brand whose design team is using an outdated web based asset management tool originally designed for photography.  Client is using this tool to manage both digital and physical assets that is used by designers for reference.  Client wants an ultra secure web app that improve user experience making assets easier to find and also improve current request/ fulfillment system.

users don't always know how to ask

Client sent mock-up of designs which they believed suited their needs.  The problem was because they were so used to using an existing tool, the mock-up just looked like a re-skin of current app.  What they needed was a entirely different type of tool built from ground up to suit their needs.

After interviewing users and mapping out the use cases and user journeys, I discovered they what they wanted was more of a "digital library" where users could checkout/ return items and advanced search features so that they could find things using natural language instead of over-categorization with the ability to add keywords over time to enhance search function.  Another important feature is the ability to create folders of saved items in order to share with other users or for future reference.

list of features
  • Advanced search features.  Instant search.  Synonyms.  Typo errors.
  • Advanced security.  2 factor authentication from unknown IP address
  • Admin dashboard for archive manager to track physical location of items
  • Create folders of saved items.  Share folders with other users

Design solution

Self hosted web app on AWS that support features and functionality as requested by client.  90% use on desktop but designed as responsive for mobile as well.

Checkout flow

Similarity of e-comm checkout and order page for user.  Dashboard for manager to track items and order fulfillment.

Algolia Search

Advanced search features based on keywords and natural language.  Ability to add/ edit keywords by all users to increase relevancy

2 Factor Authentication/ Twilio

Send code by SMS/ email when login from unknown IP

  • Home page grid view

    All items in inventory can be searched and requested for checkout

  • Create folders

    Folders can be created to collect items for sharing or future reference

  • Admin dashboard

    Archive manage can see order requests & track location of items.

Hi-fidelity prototype

I used XD to build hi-fidelity prototype. At the client's request, we skipped wireframing because of commitment to style guide and pattern library in advance. This allowed us to save a lot of time to use prototype in lieu of wireframe. The product went thru 46 iterations to get to the final design for MVP as features were added and removed througout the process. The product was prioritized desktop use but designed as repsonsive for mobile.
